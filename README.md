# Hamza Arshad

Full-stack engineer, 10+ years. JavaScript, Go, Node.js.

I spend most of my time these days on **NFC authentication** — building the backend systems that verify whether a physical product is genuine when someone taps it with their phone. Think luxury goods, artwork, wine bottles. The tag on the product talks to the phone, the phone hits my server, the server does the cryptography. That's the part I build.

I designed and built the NFC infrastructure behind [tethered.services](https://tethered.services) and [tetheredart.com](https://tetheredart.com) as a founding engineer. Both run NTAG 424 DNA with AES-CMAC verification in production.
